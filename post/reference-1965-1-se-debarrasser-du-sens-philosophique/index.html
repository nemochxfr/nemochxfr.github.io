1965 / 1 - ∞
Peinture acrylique sur tissu
196 x 135(cm^2)
Roman Opalka(1931–2011)

《1965 / 1 - ∞》是由欧帕卡不停写数字产生的作品。直到2011年8月6日去世时，他花费了将近50年的时间写满了233块画布，他写下的最后一个数字是5607249。他最后的作品近乎纯白。
为什么《1965 / 1 - ∞》是伟大的？**或许不需要更多的解释，仅仅是因为真的这么做了，它就足够伟大。**明明应该是什么哲学概念都没有的行为，却是一件伟大的观念艺术作品，这非常有趣。

艺术的表现力

艺术必须有本身的表现力。
《1965 / 1 - ∞》拥有极强的表现力，有多种方式可以让事物拥有表现力，比如高超的技巧，巨大的体积，昂贵的造价等等，这些表现力是存粹的，无论它是不是艺术作品都拥有强大的力量让人感到震撼。本作品的表现力来自于极致的重复。因此即使他最后的作品是纯白的，他的作品也散发出强烈的张力。这是由纯粹的工时累计获得的特殊质感。

哲学思想是逐渐产生的

我看过的关于《1965 / 1 - ∞》和欧帕卡的文章都喜欢说他探索时间，慢性自杀，拥有“XX主义观点”等等，然后他的过去寻找找联系，我认为这样理解艺术品的方式是错误的，很多时候这是过度解读，但好像大家都喜欢这样。在佛教中，拥有“执笔绘佛亦是修行”的概念。也就是只要持续做一件事情，你终究会悟出世间的哲理。欧帕卡只是真的这样一生，就拥有了深刻的意义。当然他的思考也通过书写规则的改变表现在了他的画上。

《1965 / 1 - ∞》给让我触动的就是他的纯粹。艺术作品完全可以是简单的东西，不需要故意往哲学上去靠拢，只是顺其自然就可以了。


在艺考期间，我们常常开玩笑把白纸称作《雪山白凤凰》，以此来调侃似乎只要编造一个好的概念，取个好的名字，艺术就可以这样诞生。
似乎总有这样一种趋势：艺术所表达的内涵，比艺术品本身更重要。因此大家都想做新的东西，都去做所谓的“观念艺术”，但这实际上是投机和欺骗。很多所谓的“观念艺术家”，只不过是一群投机分子。
我并没有在否定哲学与构思的重要性，但是实际上许多人除了有一个想歪点子的头脑之外，什么也不会。很如果观点本身比艺术品更有说服力，更吸引人，那为什么不去做一个作家呢？这些说明永远都应该是艺术品的补充。
